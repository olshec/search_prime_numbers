
Исследованы алгоритмы поиска простых чисел Вольстенхольма, 
Вифериха и поиска суперпростых чисел на бесконечном множестве чисел.

Разработаны программные реализации вышеуказанных алгоритмов на языке 
C++.

Для решения данных задач были использованы средства, позволяющие производить 
расчёты с  произвольной точностью. В качестве такого средства в работе 
использована библиотека GMP, средство для работы с длинной арифметикой.
